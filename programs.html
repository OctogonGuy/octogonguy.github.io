Java

For these programs, you need Java installed on your computer. It's an executable, so your computer may give you a warning when you open it for the first time. You should hopefully be able to open any of these .jar files by just double clicking them. If you are using Windows, it may not work for some reason, and you may need to open it through the command prompt. To do this, perform the following steps:

  1. Open Command Prompt
  2. Type "cd [directory to the .jar file]" and press Enter
  3. Type "java -jar [filename]" and press Enter

You may be able to fix it permanently it by performing the following steps:

  1. Locate javaw.exe. By default, this is located in the directory "C:\Program Files\Java\jre-[version number]\bin\".
  2. Open Command Prompt
  3. Type "ftype jarfile=[path to javaw.exe] -jar "%1" %*" and press Enter
