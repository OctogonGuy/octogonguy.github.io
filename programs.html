Java

For these programs, you need Java installed on your computer. You should hopefully be able to open any of these .jar files by just double clicking them. However, if you are using a Windows computer, it may not work for some reason. If that is the case, you may need to open it through the command prompt. If you do not know how to do this, perform the following steps:

  1. Open Command Prompt
  2. Type "cd [directory to the .jar file]" and press Enter
  3. Type "java -jar [filename]" and press Enter

You may be able to fix it permanently it by performing the following steps:

  1. Locate javaw.exe. By default, this is located in the directory "C:\Program Files\Java\jre-[version number]\bin\".
  2. Open Command Prompt
  3. Type "ftype jarfile=[path to javaw.exe] -jar "%1" %*" and press Enter
